# Foldup 3D Printer

<img width="2560" height="1440" alt="image" src="https://github.com/user-attachments/assets/618be12e-8279-407f-aaf3-1fe1eca2d280" />

This is my foldup, scissor lift, cross-cantilever 3D printer for Open Sauce 2026! It folds up into the size of a small filament box, coming in at just 200x200x75mm in form factor, and has a build area of nearly 80x80x95mm! It is a fully unique 3D printer that utilizes a Positron hotend and a custom HGX Lite bowden extruder and all of it runs off of a simple barrel-jack connector. 

## Custom Features
- Unique scissor-lift, cross-cantilever motion system
- Fits in a filament box and folds up into a 200x200x75mm form factor
- Has a build area of almost 80x80x95mm
- Runs completely off a portable barrel jack connector
- Offline printing using a MicroSD card
- Fast and efficient positron hotend with a custom HGX Lite bowden extruder
- Supports Marlin or Klipper firmware with the BTT SKR MINI E3 V3.0 motherboard
- Looks insanely cool so you can flex at hackathons
